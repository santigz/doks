## Un poco de _responsive_ Un diseño _responsive_ significa que adapta la UI al tamaño de la pantalla (móviles, etc.) Para esto se usan las **media queries**: ```css :root { font-size: 0.75em; // Pantalla pequeña } @media (min-width: 800px) { :root { font-size: 0.875em; // Pantalla mediana } } @media (min-width: 1200px) { :root { font-size: 1em; // Pantalla grande } } ```
